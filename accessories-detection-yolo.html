Accessories Detection YOLO

Real-Time Restricted Accessories Detection System

A real-time computer vision system using a custom-trained YOLO model to detect restricted accessories such as earbuds, vape, hat, etc from live camera input. The system provides confidence-based detection, FPS monitoring, and visual alert overlays.

YOLO Model
Real-Time Detection

Technologies Used

Python YOLO OpenCV Computer Vision Deep Learning

Key Features

  • Custom YOLO object detection
  • Live camera inference
  • Restricted accessory alerts
  • Bounding box visualization
  • Confidence-based filtering
  • FPS monitoring

Accessory Detection

Detects restricted accessories at office entrances using a dedicated camera, providing real-time bounding boxes and confidence scores for clear identification.

  • Earbuds detection
  • Confidence score display
  • Real-time bounding boxes
  • Single object tracking

Alert System

Triggers a real-time visual warning when prohibited accessories are detected, supporting workplace compliance and entry control.

  • Multiple object detection
  • Visual warning overlay
  • High-visibility alert UI
  • Real-time FPS display
