Accessories Detection YOLO

Real-Time Restricted Accessories Detection System

A real-time computer vision system using a custom-trained YOLO model to detect restricted accessories such as earbuds and glasses from live camera input. The system provides confidence-based detection, FPS monitoring, and visual alert overlays.

Accessory Detection

Detects restricted accessories such as earbuds with bounding boxes and confidence percentage in real-time.
