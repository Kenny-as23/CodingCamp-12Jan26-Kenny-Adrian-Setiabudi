Game Top-Up Website

Laravel-Based Game Top Up Platform

A web-based game top up platform that allows users to select games, choose top up packages, and complete transactions through a clean, structured, and user-friendly system.

Laravel Framework
MVC Architecture
CRUD System

Technologies Used

Laravel PHP Blade CSS JavaScript MySQL XAMPP

Key Features

  • Dynamic game listing from database
  • Multiple top up packages per game
  • Input validation for User ID & Server
  • Automatic price calculation
  • Structured transaction handling
  • Ready for payment gateway integration

Top Up Package

Users can select from various top up packages for their chosen game. Each package displays the amount of in-game currency and its price.

  • Dynamic package listing from database
  • Package details (currency & price)
  • Package selection interface
  • Bookmarked packages
