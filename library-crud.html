Library CRUD System

Web-Based Library Management System

A web-based library management system designed to manage book data and borrowing processes in a structured, secure, and user-friendly way. Supports role-based authentication (Admin & User).

Admin & User Roles
CRUD System
MySQL Database

Technologies Used

PHP MySQL HTML CSS JavaScript XAMPP

Key Features

  • Role-based authentication (Admin & User)
  • Book management with cover upload
  • Borrow & return system with validation
  • Borrowing history & status tracking
  • Search by title or author
  • Responsive dashboard UI

Admin Panel

Admin dashboard to manage books, users, and borrowing transactions. Designed for efficient data control and validation.

  • Manage book data (CRUD)
  • Upload & manage book covers
  • User role management
  • Borrowing & return validation

User Panel

User-friendly interface allowing users to browse, search, and borrow books independently.

  • Browse & search books
  • Borrow available books
  • View borrowing history
  • Status tracking (Returned / Late)
