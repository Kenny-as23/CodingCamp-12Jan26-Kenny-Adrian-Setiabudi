AI Face Emotion Detection

Real-Time Facial Emotion Recognition System

A real-time AI-powered web application that detects human facial emotions using computer vision and deep learning. The system supports live camera detection and image upload analysis, providing visual emotion overlays, confidence scores, and real-time statistics.

This project also integrates a heuristic-based mental condition monitoring panel that estimates indicators such as mood, stress, anxiety, and depression risk. All processing runs entirely on the client-side without a backend server.

Technologies Used

Key Features

  • Real-time face detection using webcam
  • Facial emotion recognition with confidence scores
  • Emotion visualization with bounding boxes and overlays
  • Image upload analysis mode
  • Real-time FPS, face count, and AI status monitoring
  • Mental condition heuristic indicators (mood, stress, anxiety)
  • Fully client-side AI processing (no backend required)

Real-Time Camera Detection

Live camera-based facial emotion detection with animated visual overlays, emotion labels, confidence bars, and real-time statistics.

  • Live webcam emotion detection
  • Bounding boxes & emotion labels
  • Confidence & FPS tracking
  • Animated visual feedback

Image Upload & Analysis

Image-based emotion detection allowing users to upload photos and analyze facial expressions without activating the camera.

  • Upload image for emotion analysis
  • Face detection on static images
